## 前置条件

1. **信托财产**：须为 BVI 公司的股权。非 BVI 公司的股权不能直接作为 VISTA 信托的信托财产
2. **受托人**：须为 BVI 持牌信托公司
3. **委托人**：将股权转移至信托的自然人或法人
4. **受益人**：享有信托利益的自然人（通常为家庭成员）或慈善目的

## 设立步骤

### 第一步：架构设计

与 BVI 法律顾问和信托公司共同设计信托架构：

- 确定委托人和受益人
- 确定"指定人"（Appointor）——通常为委托人本人
- 确定董事任免权安排
- 设计"介入事件"（Intervention Events）触发条件

### 第二步：设立 BVI 公司

设立作为信托财产的 BVI 商业公司。该公司的股权将被转移至信托中。公司章程应与信托架构协调。

### 第三步：起草信托契约

VISTA 信托契约是核心法律文件，须明确规定：

- 受托人的权利义务范围
- 董事任免权的归属
- "介入事件"清单（在何种情况下受托人可以或必须介入）
- 指定人的继任安排
- 受托人的报酬
- 信托的终止条件

### 第四步：签署与交割

- 委托人签署信托契约
- 受托人签署信托契约
- 委托人将 BVI 公司股权转让给受托人
- 向公司注册处更新股东名册

### 第五步：后续安排

- 通知 BVI 金融机构（如有需要）
- 确保董事知悉 VISTA 结构
- 更新相关合同文件

## 时间线

| 阶段 | 时间 |
|------|------|
| 架构设计 | 1-4 周 |
| 设立 BVI 公司（如尚未设立） | 1-3 周 |
| 起草与审阅信托契约 | 2-4 周 |
| 签署与交割 | 1-2 周 |
| **总计** | **约 1-3 个月** |

## 关键注意事项

- VISTA 信托仅适用于 BVI 公司股权
- 信托契约中的"介入事件"条款需精心设计
- 指定人继任安排至关重要
- 应考虑 VISTA 与其他工具（如 PTC）的组合使用
